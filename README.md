# Дашборд продаж Kaspi

Сайт для ежедневного отслеживания продаж твоего Kaspi-магазина: сколько продано всего и по каждому товару, за любой период, с автоматическим обновлением данных каждую ночь. Создан, чтобы понимать, как растут или падают продажи — особенно полезно для оценки эффективности рекламы (например, в TikTok).

## Что внутри

- **`backend/`** — программа, которая раз в сутки забирает заказы из Kaspi API и считает статистику. Развёртывается на Render.
- **`frontend/`** — сам сайт с графиками, который ты открываешь в браузере. Развёртывается на Vercel.
- **`docs/DEPLOY.md`** — пошаговая инструкция, как всё это запустить, если ты не программист.

## Как это работает

1. Backend каждую ночь в 03:00 ходит в Kaspi API и забирает все заказы за последние дни
2. Для каждого заказа подгружается состав (какие товары и в каком количестве)
3. Всё сохраняется в базу данных (Postgres)
4. Сайт обращается к backend и строит из этих данных графики и таблицы — выбор любого периода, сравнение "сегодня против вчера", разбивка по каждому товару

## С чего начать

Открой **`docs/DEPLOY.md`** — там всё расписано по шагам, от получения токена Kaspi до открытия готового сайта в браузере.
